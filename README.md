## ✨ Funcionalidades
- 📂 Organização de **dados educacionais** (matrículas, notas, frequência, logs de LMS).  
- 🧹 **Limpeza e padronização** de dados.  
- 📈 **Indicadores de aprendizagem**: engajamento, evasão, desempenho.  
- 🤖 **Machine Learning** para prever riscos de reprovação/abandono.

- 📊 **Dashboards automáticos** para acompanhamento.
